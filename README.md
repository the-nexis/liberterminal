# >_ Liberterminal

A decentralized, serverless community hub built on the Nostr protocol. 

Liberterminal is a censorship-resistant forum designed for sharing ideas, media, and deep discussion without relying on centralized servers or databases.

## Tech Stack
* **Frontend:** React + Vite
* **Styling:** Tailwind CSS v4
* **Protocol:** Nostr (`@nostr-dev-kit/ndk`)
* **Auth:** NIP-07 Browser Extensions (tested with nos2x)

## Running Locally

1. Clone the repository:
   ```bash
   git clone [https://github.com/YOUR_USERNAME/Liberterminal.git](https://github.com/YOUR_USERNAME/Liberterminal.git)
   cd Liberterminal

2. Install dependencies:
   ```bash
   npm install

2. Run locally:
   ```bash
   npm run dev
